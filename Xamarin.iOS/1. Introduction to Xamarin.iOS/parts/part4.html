Introduction to Xamarin.iOS

Duration

10 minutes

Lab Goals

In this exercise, we will be adding the business logic to our Tip Calculator by handling the tap event on the button and calculating the tip amount. This logic is the Model for our application (we created the View in the prior exercise). We could locate this logic in a physically separate class, but given the simplicity, it makes more sense to just place it right in the View Controller source file.

Required assets

The provided Part 04 Resources folder contains a subfolder named Completed with a solution you can use to check your work.

High-level steps

Use the following high-level steps to complete the exercise, step-by-step instructions are provided below that if you need any guidance.

  1. Handle the TouchUpInside event on the calcButton.
  2. In the event handler, get the entered amount from the totalAmount UITextField and multiply it by 20%.
  3. Display the resulting value as a currency amount in the resultLabel.

Steps

Implement the Tip Calculator

In this section, we will add the logic for the Tip Calculator. You can use either Xamarin Studio or Visual Studio - the instructions are identical for both.

  1. At the bottom of your ViewDidLoad method, add an event handler for the TouchUpInside method on your calcButton field.
    2. Since we are using fields local to ViewDidLoad, you will need to add an inline delegate handler using either an anoymous delegate, or a lambda expression. If you would prefer to use a traditional delegate method handler, move the field definitions for your three controls to the class so you can access them from the separate method. The lab solution shown below will use a lambda.
  2. In your method handler, retrieve the current value of the totalAmount field using the Text property.
  3. Convert the string value (it's text) into a double using Double.TryParse which is a standard .NET Framework feature.
  4. Multiply the result by 0.2 to get 20% of the total - note that it may be zero if the field did not convert to a double, or you enter zero into the field. In either case, the tip will still be zero.
  5. Finally, store the resulting text into the resultLabel field by setting the Text property. Go ahead and format it using string.Format so it matches the default text we set earlier.

    6. Show Code 

    public override void ViewDidLoad()
{
    base.ViewDidLoad();
    ...
    
    View.AddSubviews(new UIView[] { totalAmount, calcButton, resultLabel });

    calcButton.TouchUpInside += (s, e) => {
        double value = 0;
        Double.TryParse(totalAmount.Text, out value);
        resultLabel.Text = string.Format("Tip is {0:C}", value * .2);
    };
}
