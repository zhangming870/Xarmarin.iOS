Getting started with the iOS Designer

Duration

5 minutes

Tip: If you are doing this exercise live in a session, make sure to make good use of the instructor, they are online to answer any questions you have!

Goals

The primary goal of this lab is to display a second View Controller using segues.

Required assets

This lab is a continuation of the previous exercise. You may use your project from exercise 04 or open the completed project in the Part 04 Resources folder. There is a completed version of the exercise in the Part 05 Resources folder if you'd like to compare your solution when you are finished.

Challenge

Update your application to display the About screen using a Segue

  1. Remove the code to launch the About screen programmatically.
  2. Add a modal segue from the About button to the About screen.

Steps

Below are the step-by-step instructions to implement the exercise.

Remove the code to launch the About screeen

  1. Open ViewController.cs and delete the partial method buttonAbout_TouchUpInside.
  2. Open Main.storyboard.
  3. Select the About button.
  4. In the Properties pane, go to the Events section and clear the reference for Up Inside.

Add a Segue to the About screen

  1. Control-drag from the About button to the About View Controller and select a Modal Segue.
  2. Run the application.

Summary

In this lab you displayed a second screen using a modal segue.
